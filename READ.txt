Welcome to Task Handler application!

In order to start please proceed following instructions:

- create folder on your local computer ( for instance taskHandler )
- clone latest application from GitHub https://github.com/Maksim1990/Task-Handler.git
- add new domain to your local server (all application functionality were tested on the latest version of OpenServer.
OpenServer settings:
1.Settings->Domains
2. Add new domain with any name, for example "taskHandler", domain folder - choose Public folder in downloaded application.
3.Press Save.)
- Go to the taskHandler/ and enjoy application.



This application were tested under following conditions:
Windows 10 Pro, Processor(R) Dual-Core CPU T4500 @ 2.30GHz 2.30 GHz,
RAM 2GB,
System Type: 32-bit,
OpenServer cross-platform web server solution
Apache-2.4
PHP-5.6
MySQL-5.5
Backend -Laravel 5 framework
